# Claude Code Agent Instructions

This repository contains personal skills for enhancing your product ownership and development workflow.

## Skills Index

| Skill | Path | Description |
|-------|------|-------------|
| Product Owner Assistant | `product-owner-assistant/` | Epic/sub-epic creation, requirements structuring, sprint planning, YouTrack MCP integration |

## How to Handle This Repo

### Adding New Skills
1. Create a new directory under `/`
2. Add a `SKILL.md` file with skill metadata and instructions
3. Update the **Skills Index** in this file
4. Update `README.md` with the new skill

### Updating the Skills Index
When adding or modifying skills, update both:
- The **Skills Index** table in this file
- The Skills section in `README.md`

### Invoking Skills
Reference skills by their directory name in conversation (e.g., `/product-owner-assistant`).
